# 🎯 Elite Shooter Pro (Precision Edition)

这是一款基于 **HTML5 Canvas 逻辑**与 **CSS3 硬件加速**开发的轻量级弹射小游戏。专为移动端与 PC 端双端适配优化，具备极高的物理交互灵敏度与精准的碰撞检测机制。

## 🚀 核心特性

* **全方位弹射系统 (360° Ballistics)**：取消了传统的拉弓限制，支持 360 度任意方向拉伸，通过反向拉力矢量算法实现精准投射。
* **零延迟触控响应 (Ultra-Responsive)**：针对手机端深度优化，移除点击死区与预设门槛，支持 1 像素级的微操触发。
* **精准计分逻辑 (Single-Hit Lock)**：引入碰撞状态锁机制，确保小球在单次飞行路径中仅触发一次有效计分，杜绝重复计分 Bug。
* **平滑视觉体验 (Smooth Animation)**：采用 `requestAnimationFrame` 配合 CSS `translate3d` 硬件加速，确保靶子在不同分辨率设备上均能保持恒定的丝滑滑动感。
* **自适应布局 (Responsive Design)**：自动识别横竖屏切换，动态调整弹弓位置与靶子运动轨迹，完美适配各种手机浏览器。

## 🛠️ 技术栈

* **框架**: 纯原生 JavaScript (Vanilla JS)
* **样式**: Tailwind CSS (CDN) + 自定义 CSS3 动画
* **矢量渲染**: Inline SVG (用于绘制动态拉伸的弓弦)
* **算法**: 
    * 基于胡克定律模拟的拉伸反馈
    * 基于 AABB (Axis-Aligned Bounding Box) 的动态碰撞检测
    * 抛物线运动轨迹模拟

## 🎮 操作指南

1.  **启动**: 点击屏幕中央的 **START** 按钮开启任务。
2.  **瞄准**: 触摸/点击蓝色小球，向**目标相反方向**拉动（拉得越远，弹力越强）。
3.  **射击**: 松开手指/鼠标，小球将根据拉力矢量飞出。
4.  **得分**: 击中不同颜色的分值区域获得对应分数，注意避开扣分区域！
